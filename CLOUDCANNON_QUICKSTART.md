# CloudCannon Quick Start Guide

## 5-Minute Setup

### Step 1: Push to Git (2 minutes)
```bash
cd "/Users/ihnatovvladgmail.com/Downloads/SaunaPageSite-main 3"
git add .
git commit -m "Add CloudCannon configuration"
git push origin main
```

### Step 2: Create CloudCannon Account (1 minute)
1. Go to [CloudCannon.com](https://cloudcannon.com)
2. Click "Get Started Free"
3. Sign up with email or GitHub account
4. Confirm email

### Step 3: Connect Your Repository (2 minutes)
1. In CloudCannon, click "Create Site"
2. Select your Git provider (GitHub/GitLab/Bitbucket)
3. Authorize CloudCannon
4. Select `SaunaPageSite-main` repository
5. Click "Build Site"

## You're Done! 🎉

Your site is now live in CloudCannon. Start editing:

1. Click "Editor" to open the visual editor
2. Click any text or image to edit
3. Changes automatically save to Git

## Editing Tips

### Keyboard Shortcuts
- `Ctrl+B` - Bold text
- `Ctrl+I` - Italic text
- `Ctrl+U` - Underline text
- `Ctrl+Z` - Undo
- `Ctrl+Shift+Z` - Redo

### Format Text
- **Bold**: Select → Click bold icon
- *Italic*: Select → Click italic icon
- ~~Strikethrough~~: Select → Click strikethrough
- [Links](http://example.com): Select → Click link icon

### Image Management
- Click image → Upload new one
- Supported formats: JPG, PNG, GIF, WebP
- Max file size: 100MB (usually smaller is better)
- CloudCannon auto-optimizes images

## Troubleshooting

### Changes Not Showing?
→ Clear browser cache (Ctrl+Shift+Del)
→ Hard refresh (Ctrl+F5)
→ Wait 30 seconds for build

### Image Upload Failed?
→ Check file size (under 100MB)
→ Use JPG, PNG, GIF, or WebP
→ Ensure good internet connection

### Can't Log In?
→ Check email/password
→ Reset password if forgotten
→ Contact support@cloudcannon.com

### Content Looks Wrong?
→ Check styling in CSS
→ Verify HTML structure
→ Check for typos in content

## Team Collaboration

### Add Team Members
1. CloudCannon settings → Team
2. Click "Invite Team Member"
3. Enter email address
4. Set permission level
5. Send invite

### Permission Levels
- **Editor**: Can edit content, create pages
- **Viewer**: Can view but not edit
- **Admin**: Full control of site

### Common Questions

**Q: Will editing break the design?**
A: No! You can only edit content marked with `editable` class. Design is protected.

**Q: Do I need to know HTML?**
A: No! CloudCannon's visual editor handles it all.

**Q: Can I undo mistakes?**
A: Yes! Use Undo button or rollback from history.

**Q: Where are my changes saved?**
A: In Git repository automatically. You have full version history.

**Q: Can multiple people edit at once?**
A: Yes! CloudCannon handles conflicts automatically.

**Q: Will my SEO be affected?**
A: No! Actually improves with regular updates.

**Q: What if I break something?**
A: Easy rollback to previous version anytime.
